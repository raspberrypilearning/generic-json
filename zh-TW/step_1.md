JSON是一種用於儲存和共享數據的格式。 JSON（發音為Jason）是JavaScript Object Notation的縮寫，但它不僅限用於JavaScript。

JSON是一種可以在代碼中使用的純文字格式，人們可以很容易地閱讀。

```json
{
  "名稱": "食人魔",
  "大小": 90, 
  "力量": 86, 
  "智能": 12, 
  "魔力": 0
}
```

JSON **物件** 以大括號 `{}`列出所有關鍵值來表示。

值在中括號`[]`裡可用來表示 **列表** ：

```json
{
  "名稱": "食人魔",
  "大小": 90,
  "力量": 86,
  "智能": 12,
  "魔力": 0,
  "武器": ["木棍", "石頭", "骨頭"]
}
```
