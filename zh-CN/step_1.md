JSON 是用于储存和分享数据的一种格式。JSON（念 Jason）指 JavaScript 对象表示法，但其不只用于 JavaScript。 

JSON 是一种文本格式，可在代码中使用且十分方便人们读取。 

```json
{
  "name": "Ogre",
  "size": 90,
  "power": 86,
  "intelligence": 12,
  "magic": 0
}
```

JSON **对象**指花括号 `{}` 内部的一列键值对。

值也可以是方括号 `[]` 内部的一个**列表**。

```json
{
  "name": "Ogre",
  "size": 90,
  "power": 86,
  "intelligence": 12,
  "magic": 0,
  "weapons" : ["club", "rock", "bone"]
}
```
